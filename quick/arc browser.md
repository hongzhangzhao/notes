

# 1 第一层

基础功能

- ctrl + t = 新建页面
- ctrl + shift + + = 新建分屏页面
- ctrl + r = 刷新页面
- ctrl + shift + t = 打开刚才关闭的页面
- ctrl + shift + n = 打开无痕浏览
- ctrl + tab = 切换页面

# 2 第二层

全屏/分屏/预览/小窗

- ctrl + s = 打开或关闭侧边栏
- alt + 鼠标左键 = 点击连接快速分屏
- shift + 鼠标左键 = 点击连接快速预览, ESC快速退出


# 3 第三层

标签管理/高度自定义

- ctrl + d = 将网页收藏到space下


# 4 第四层

空间(space)/配置文件(profile)

- ctrl + alt + 左右方向键 = 切换space
- alt + 数字键 = 切换space
